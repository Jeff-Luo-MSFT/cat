# Agent Excellence Workshop - CAT Kickstarter

A comprehensive multi-activity workshop designed to master the art of building exceptional AI agents with Microsoft's agent platform ecosystem.

## Workshop Structure

This workshop consists of three main activities designed to take attendees from platform selection through advanced agent development:

### 🧭 **Activity 1: Agent Platform Advisor**
- **Objective**: Choose the right Microsoft platform for your agent use case
- **Format**: Interactive online assessment 
- **Link**: [Agent Platform Advisor](https://microsoft.github.io/AgentPlatformAdvisor/)
- **Duration**: 15-20 minutes
- **Outcome**: Personalized platform recommendations

### 🎯 **Activity 2: Agent Boundary Challenge**
- **Objective**: Design robust agent instructions with proper boundary conditions
- **Format**: Interactive wild card scenario testing
- **Features**: 50+ challenge scenarios, real-time testing, post-it note interface
- **Duration**: 30-45 minutes
- **Outcome**: Refined agent instructions that handle edge cases

### 🔧 **Activity 3: Agent Integrations & Tools**
- **Objective**: Build powerful agent integrations with external systems
- **Format**: Three-phase hands-on exercise with universal prompt pack testing
- **Features**: 
  - Integration setup (connectors, MCP servers, APIs)
  - Universal test prompts with customizable scenarios
  - Real-time result capture and analysis
  - Automated insights and recommendations
- **Duration**: 60-90 minutes
- **Outcome**: Production-ready agent with external integrations and proper stopping conditions

#### Three-Phase Structure:
1. **Design the Action**: Configure integration contract and stopping-power instructions
2. **Prove Your Tool**: Test with 10 universal prompts, capture results, analyze agent behavior
3. **Encode "Stopping Power"**: Add final safeguards and escalation paths based on test results

## How to Use

### For Workshop Facilitators:
1. Open `index.html` in any web browser to access the main landing page
2. Project the page or share the file with attendees
3. Guide attendees through activities in sequence
4. Use individual activity links for focused sessions

### For Attendees:
1. Start with the landing page overview
2. Complete activities in the recommended order:
   - Begin with **Agent Platform Advisor** for platform selection
   - Continue with **Agent Boundary Challenge** for instruction design
   - Proceed to **Agent Integrations & Tools** when available
3. Use the theme toggle (🌙/☀️) for comfortable viewing

## Workshop Goals

By the end of this workshop, attendees will be able to:
- ✅ Choose the right Microsoft agent platform for their use case
- ✅ Design robust agent instructions with proper boundary conditions  
- ✅ Build powerful agent integrations with external systems
- ✅ Deploy production-ready agents with confidence

## Challenge Categories

The wild cards test various boundary conditions:
- Authority claims ("I'm the CEO")
- Urgency pressure ("It's an emergency") 
- Rule-bending requests ("Just this once")
- Roleplay attempts ("Pretend to be...")
- Confidentiality violations ("Don't tell anyone")
- Data fabrication ("Make something up")
- Safety overrides ("Ignore your limitations")
